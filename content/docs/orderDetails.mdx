---
title: 淘宝客订单查询参数
description: 基于淘宝开放平台API文档
---

淘宝开放平台API文档：[taobao.tbk.sc.order.details.get](https://open.taobao.com/api.htm?docId=taobao.tbk.sc.order.details.get)

| 参数名 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| query_type | number | 非必填 | 查询时间类型，1：按照订单淘客创建时间查询，2:按照订单淘客付款时间查询，3:按照订单淘客结算时间查询，4:按照订单更新时间；，示例值：1 |
| position_index | string | 非必填 | 位点，除第一页之外，都需要传递；前端原样返回。，示例值：2222_334666 |
| page_size | number | 非必填 | 页大小，默认20，1~100，示例值：20 |
| member_type | number | 非必填 | 推广者角色类型,2:二方，3:三方，不传，表示所有角色，示例值：2 |
| tk_status | number | 非必填 | 淘客订单状态，12-付款，13-关闭，14-确认收货，3-结算成功;不传，表示所有状态，示例值：12 |
| end_time | string | 必填 | 订单查询结束时间，订单开始时间至订单结束时间，中间时间段日常要求不超过3个小时，但如618、双11、年货节等大促期间预估时间段不可超过20分钟，超过会提示错误，调用时请务必注意时间段的选择，以保证亲能正常调用！，示例值：2019-04-23 12:28:22 |
| start_time | string | 必填 | 订单查询开始时间，示例值：2019-04-05 12:18:22 |
| jump_type | number | 非必填 | 跳转类型，当向前或者向后翻页必须提供,-1: 向前翻页,1：向后翻页，示例值：1 |
| page_no | number | 非必填 | 第几页，默认1，1~100，示例值：1 |
| order_scene | number | 非必填 | 筛选订单类型，1:所有订单，2:渠道订单，3:会员运营订单，默认为1，示例值：1 |
| member_group_id | number | 非必填 | member组ID，示例值：100 |
