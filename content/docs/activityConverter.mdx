title: 淘宝客官方活动转链参数
description: 基于淘宝开放平台API文档
---

淘宝开放平台API文档：[taobao.tbk.sc.activity.info.get](https://open.taobao.com/api.htm?docId=taobao.tbk.sc.activity.info.get)

| 参数名 | 类型 | 是否必填 | 描述 |
| --- | --- | --- | --- |
| activity_material_id | string | 必填 | 官方活动会场ID，从淘宝客后台"我要推广-活动推广"中获取，示例值：123 |
| relation_id | number | 非必填 | 渠道关系id，示例值：123 |
| union_id | string | 非必填 | 自定义输入串，英文和数字组成，长度不能大于12个字符，区分不同的推广渠道，示例值：demo |
