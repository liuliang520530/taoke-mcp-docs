---
title: 拼多多PID查询参数
description: 基于拼多多开放平台API文档
---

拼多多开放平台 API 文档：[pdd.ddk.oauth.goods.pid.query](https://open.pinduoduo.com/application/document/api?id=pdd.ddk.oauth.goods.pid.query)

| 参数名    | 类型   | 是否必填 | 描述                              |
| --------- | ------ | -------- | --------------------------------- |
| page      | number | 非必填   | 返回的页数                        |
| page_size | number | 非必填   | 返回的每页推广位数量              |
| pid_list  | array  | 非必填   | 推广位列表，例如：`["60005_612"]` |
